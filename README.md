# Underwater Image Enhancement using Deep Learning

## 📌 Overview
This project focuses on enhancing underwater images by addressing issues such as color distortion, low contrast, and poor visibility. A deep learning-based approach using transfer learning is employed to restore image quality and improve visual clarity.

---

## ⚙️ Methodology

### 1. Input Processing
- Collected underwater images with degraded visibility and color distortion

### 2. Model Design
- Utilized pretrained Convolutional Neural Networks (CNNs)
- Applied transfer learning for feature extraction and enhancement

### 3. Image Enhancement
- Restored color balance
- Improved contrast and visibility in turbid conditions

---

## 🛠️ Tech Stack
- Python  
- TensorFlow  
- OpenCV  
- Matplotlib  

---

## 📊 Results
- Significant improvement in image clarity and contrast  
- Effective color restoration in degraded underwater images  
- Outperformed traditional enhancement techniques on benchmark datasets

## 🎯 Key Contributions
